# QuickBite Django REST API

## Description
This project is a Restaurant Menu Management API built using Django REST Framework and MongoDB.

## Features
- Create menu item
- Get all menu items
- Get menu by ID
- Update menu
- Delete menu
- Search by restaurant
- Filter by category
- Filter by availability

## Tech Stack
- Django
- Django REST Framework
- MongoDB (MongoEngine)

## How to Run
```bash
git clone <repo>
cd quickbite-django-api
venv\Scripts\activate
cd quickbite
python manage.py runserver
